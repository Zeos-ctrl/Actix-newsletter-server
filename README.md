## Prepare for use 
In the projects root you need to make a .env file with DATABASE_URL and NEWSLETTER_DIR
set before the program works.

```
$ touch .env
$ echo "DATABASE_URL=database_url\nNEWSLETTER_DIR=newsletter_dir" >> .env

```
